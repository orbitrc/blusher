Blusher
==========

Desktop GUI Toolkit written in QML.


Install
----------

Currently Blusher default supports only path exactly /usr/lib/blusher

```sh
$ sudo make install
```

Above command will copy lib/blusher directory in the project root to /usr/lib/

------------------------------
